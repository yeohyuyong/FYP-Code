\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

This project develops a framework for assessing economic resilience under disruption using the Dynamic Inoperability Input-Output Model (DIIM) \cite{SantosHaimes2004,JinZhou2023}. The DIIM captures how sectoral disruptions propagate through input-output linkages and how the economy recovers over time. Two disruption scenarios relevant to Singapore are studied: (1)~the COVID-19 pandemic, using the 2019 input-output (IO) table aggregated to 15~sectors, and (2)~a manpower disruption scenario motivated by potential grey zone conflicts that could trigger large-scale foreign worker departures, using the 2022 IO table with 15~sectors.

In disruption management, a key question is which sectors should receive priority intervention. The DIIM can identify these ``key sectors'' by simulating the full disruption and ranking sectors by economic loss, but this approach requires crisis-time data (initial inoperability~$q(0)$, demand perturbation~$c^{*}$) that may not be available during the early stages of an event. This motivates the search for simplified alternatives that use only the IO table.

This study evaluates five simplified methods against the DIIM benchmark through Monte Carlo simulation across multiple top-$k$ values. Performance is measured by the close rate: the fraction of trials where a simplified method achieves at least 80\% of the DIIM's loss reduction. Results vary with the number of targeted sectors~$k$. At $k \geq 10$, all five methods achieve close rates above 92\% in both scenarios. At smaller~$k$, performance drops: for the COVID scenario at $k = 5$, PCA $\times\, x_i$ achieves a 64.1\% close rate while the other four methods fall to 5.7\%; for the manpower scenario at $k = 5$, all methods achieve around 45.5\%. No single structural method dominates across all conditions.
